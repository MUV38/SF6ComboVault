# SF6ComboVault

Street Fighter 6のコンボレシピを保存・検索・共有できる静的Webアプリです。

## Features

- キャラ別のコンボ保存
- 最初にキャラを選んでからコンボ作成・閲覧
- タグ付け、検索、お気に入り
- キャラ別メモと保存済みコンボのリンク
- キャラメモから公式コマンドリスト・フレーム表を確認
- ボタン選択式のコンボレシピ作成
- 方向、攻撃、OD、TC、一回転、半回転コマンドのレシピ入力
- 公式ページ風のコマンド表示
- 保存済みコンボごとの練習記録
- 1P側 / 2P側の成功率、回数、最終練習日の記録
- 苦手ポイント、練習メモ、ステータス管理
- 今日の練習リストとキャラ別練習ダッシュボード
- コンボ単体リンク、共有URL、JSONによる保存データ共有
- 取り込み時は既存データを上書きせず、重複を避けて追加
- コンボ作成、保存済みコンボ、キャラメモ、練習ページの切り替え
- PC、タブレット、スマホ対応のレスポンシブUI

## Data Sharing

保存データ全体は共有URLまたはJSONファイルで共有できます。共有URLを開いただけでは自動取り込みせず、画面上の読み込み操作で既存データに追加します。

URLが長くなる場合はJSON書き出しを使うと扱いやすくなります。

## Publish With GitHub Pages

GitHubのリポジトリ設定で、Pagesの公開元を `main` ブランチの `/root` にすると公開できます。
