# SF6ComboVault

Street Fighter 6のコンボレシピを保存・検索・共有できる静的Webアプリです。

## Features

- キャラ別のコンボ保存
- 最初にキャラを選んでからコンボ作成・閲覧
- タグ付け、検索、お気に入り
- キャラ別メモと保存済みコンボのリンク
- ボタン選択式のコンボレシピ作成
- 共有URLのコピーと取り込み
- 保存データ全体の共有URLコピーと取り込み
- コンボ作成ページと保存済みコンボページの切り替え
- PC、タブレット、スマホ対応のレスポンシブUI

## Publish With GitHub Pages

GitHubのリポジトリ設定で、Pagesの公開元を `main` ブランチの `/root` にすると公開できます。
